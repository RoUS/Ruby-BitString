new([val], [length]) => BitString
new(length) {|index| block } => BitString

Description

Create a new BitString object. By default, it will be unbounded and all bits clear.

Arguments

val
Array, Integer, String, or BitString. Initial value for the bitstring. If a String, the value must contain only ‘0’ and ‘1’ characters; if an Array, all elements must be 0 or 1. Default 0.
length
Integer. Length (number of bits) for a bounded bitstring. Default 1.

Exceptions

ArgumentError
Length specified but bitstring is unbounded.
RangeError
Value is a string, but contains non-binary digits.

bitstring.grow(bits, [defval], [direction]) => BitString

Description

Return a new bitstring based on the current one, grown (made longer) in one direction (toward the least significant bits) or the other. Growing an unbounded string toward the high end is a no-op.

Bits added are set to defval (default 0).

Arguments

bits
Integer Number of bits to add.
defval
Integer Value to which added bits should be set.
direction
Either BitString::HIGH_END (the default) or BitString::LOW_END. Indicates whether bits are added at the least or most significant end. Growing with BitString::LOW_END results in the bitstring being shifted left.

Exceptions

ArgumentError
bits isn‘t an integer, defval can‘t be reduced to a binary value, or direction isn‘t one of the defined values.
IndexError
bits is negative or meaningless.
RuntimeError
Can‘t grow an unbounded string at the high end.

bistring.rotate(bits) => BitString

Description

Rotate the bitstring, taking bits from one end and shifting them in at the other. Only makes sense with bounded strings.

A negative value rotates left; a positive one rotates to the right.

Arguments

bits
Integer. Number of positions to rotate left (negative) or right (positive). Bits rotated off one end are rotated in on the other.

Exceptions

RuntimeError
Can‘t rotate an unbounded bitstring.
