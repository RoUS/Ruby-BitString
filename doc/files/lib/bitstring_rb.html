Synopsis

   require 'rubygems'
   require 'bitstring'
   bString = BitString.new([initial-value], [bitcount])
   bString = BitString.new(bitcount) { |index| block }

Description

The BitString package provides a class for handling a series of bits as an array-like structure. Bits are addressable individually or as subranges.
